# common-components-lib
### 公共的类、指令、方法和组件，适用于小作坊内部使用~~~

## 开发
```
npm install

// 建议不要直接使用 cnpm 安装依赖，会有各种诡异的 bug。可以通过如下操作解决 npm 下载速度慢的问题
npm install --registry=https://registry.npm.taobao.org
```

### 本地验证
```
npm run dev
```

### 编译
```
npm run lib
```

### Lints and fixes files
```
npm run lint
```
